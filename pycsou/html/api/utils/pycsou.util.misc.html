peaks(x: numpy.ndarray, y: numpy.ndarray) → numpy.ndarray[source]¶

Matlab 2D peaks function.

Peaks is a function of two variables, obtained by translating and scaling Gaussian distributions (see Matlab’s peaks function <https://www.mathworks.com/help/matlab/ref/peaks.html>). This function is useful for testing purposes.

Parameters

  • x (np.ndarray) – X coordinates.

  • y (np.ndarray) – Y coordinates.

Returns

Values of the 2D function peaks at the points specified by the entries of x and y.

Return type

np.ndarray

Examples

import numpy as np
import matplotlib.pyplot as plt
from pycsou.util.misc import peaks

x = np.linspace(-3,3, 1000)
X,Y = np.meshgrid(x,x)
Z = peaks(X,Y)
plt.figure()
plt.imshow(Z)
